# HCMUS-GPA-bookmarlet
Tính GPA trên Portal thông qua bookmarklet.

Ấn [**vô đây**](https://ngntrgduc.github.io/HCMUS-GPA-bookmarklet) để chuyển hướng tới trang web hướng dẫn sử dụng.

### Cách tính GPA
